Room Terminal HUD Screen

A room display system for smart information and custom interface design.

Project Overview

This project is a room-based terminal/HUD style display intended to show useful information in a clean, custom format. The goal is to make a screen that feels integrated into the room rather than just placing a generic display there.

What I Focused On

I focused on display layout, smart home data presentation, room integration, and the overall design of a system that combines information, style, and function. It also connects with my interest in dashboards and custom interfaces.

Why I Built It

This project shows how I think not just about wiring and devices, but also about how users interact with systems in a real environment.
