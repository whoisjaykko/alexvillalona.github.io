Room Terminal HUD Screen

Low-voltage display and control installation project.

Project Overview

This project focused on building a room-based display/control system that required low-voltage power, clean installation planning, and integration with connected devices.

Hands-On Work Performed

Planned display placement and low-voltage power routing. Integrated the screen with connected systems for live information and control. Managed cable routing and layout to keep the installation clean and organized. Worked through system setup and troubleshooting during integration.

Real-World Value

This build strengthened installation planning, low-voltage setup, cable management, and practical system integration skills.
