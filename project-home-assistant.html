Home Assistant Smart Home

Low-voltage control and sensor system build.

Project Overview

This project involved installing and configuring low-voltage devices, sensors, and control systems throughout a custom smart home setup. The work focused on wiring, device integration, troubleshooting, and making multiple components work together reliably.

Hands-On Work Performed

Installed and configured low-voltage wiring for sensors and control systems. Built and wired automation circuits using ESP32 controllers. Troubleshot power and signal issues across multiple connected components. Routed and organized wiring for a cleaner and more serviceable setup.

Real-World Value

This project strengthened my comfort with low-voltage wiring, troubleshooting, controls, sensors, cable management, and system integration.
